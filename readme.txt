
We used curl command to send HTTP requests to server from client browser


GET REQUEST:
run "://localhost:8080/(Name of file,which we want to access)" on browser 
or
run "curl -X GET ://localhost:8080/(Name of file, which we want to access)" on terminal


PUT REQUEST:
run "curl -X PUT ://localhost:8080/(location of file, which we want to update)" on terminal


POST REQUEST:
run "curl -X POST ://localhost:8080/(location of file, which we want to upload)" on terminal


DELETE REQUEST
run "curl -X DELETE ://localhost:8080/(Name of file, which we want to delete)" on terminal
